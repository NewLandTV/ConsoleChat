# How to run

Clone this repository.

```sh
git clone https://github.com/NewLandTV/ConsoleChat.git
cd ConsoleChat
```

```sh
npm install
```

Create a config.json file.

```json
{
    "clientId": "bot client id",
    "guildId": "guild id",
    "token": "bot token",
    "yourId": "your id"
}
```

```sh
deploy
```

```sh
node .  # node index.js
```
